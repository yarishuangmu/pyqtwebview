# 说明和注意事项
# ===============
#
# 1. 系统要求:
#    - Python 3.6+
#    - 支持Qt5的操作系统 (Windows, macOS, Linux)
#    - 足够的内存运行Chromium引擎 (推荐4GB+)
#
# 2. 安装说明:
#    - Windows: pip install PyQt5 PyQtWebEngine
#    - macOS: pip install PyQt5 PyQtWebEngine (可能需要brew install qt5)
#    - Linux: sudo apt-get install python3-pyqt5 python3-pyqt5.qtwebengine (Ubuntu/Debian)
#              或 pip install PyQt5 PyQtWebEngine
#
# 3. 可能遇到的问题:
#    - WebEngine需要显卡加速支持
#    - 某些Linux发行版可能需要额外的Qt5库
#    - macOS可能需要额外配置路径
#
# 4. 替代方案:
#    - 如果PyQtWebEngine安装困难，可以考虑使用PyQt5.QtWebKit (功能较少)
#    - 也可以使用tkinter + webview库作为轻量级替代
#
# 5. 版本兼容性:
#    - PyQt5 5.15.x 是最后的开源版本
#    - 更新版本需要商业许可证
#    - 建议使用稳定的LTS版本
